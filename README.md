# GIMNOFANGE HEALTH CLUB

![GIMNOFANGE Preview](assets/img/preview.gif)

Modern and professional fitness center website, built with cutting-edge web technologies.

## 🏋️‍♂️ Features

- Modern and responsive design
- Dark/Light theme
- Smooth scrolling navigation
- Organized sections:
  - Dynamic Hero Section
  - About Us with gym history
  - Professional Services
  - Fitness Classes
  - Expert Trainers
  - Membership Plans
  - Contact Form
- SEO optimized
- High performance
- Cross-browser compatibility

## 🚀 Technologies

- HTML5
- CSS3 (with CSS variables)
- JavaScript (ES6+)
- Bootstrap 5.3
- Font Awesome 6
- Google Fonts
- WebP image optimization

## 💻 Installation

1. Clone the repository:
```bash
git clone https://github.com/your-username/gimnofange.git
```

2. Open `index.html` in your browser or use a local server.

## 🔧 Customization

### Themes
The site supports light and dark modes, controlled by:
- Theme toggle button
- System preference
- User choice persistence

### Responsive Design
- Mobile First approach
- Breakpoints for various devices:
  - Mobile: 320px+
  - Tablet: 768px+
  - Desktop: 1024px+
  - Wide: 1440px+

- Responsive hamburger menu
